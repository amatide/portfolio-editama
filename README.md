## 📦 How to Use

### Opsi 1: Langsung buka (GitHub Pages)
Buka salah satu URL live di atas — tidak perlu install apapun.

### Opsi 2: Jalankan lokal
```bash
git clone https://github.com/amatide/portfolio-editama.git
cd portfolio-editama
python3 -m http.server 8080
```
Kemudian buka `http://localhost:8080`

### Opsi 3: Deploy ulang ke GitHub Pages
```bash
chmod +x deploy-pages.sh
./deploy-pages.sh
```
